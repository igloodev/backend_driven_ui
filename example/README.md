# Backend-Driven UI Example

Example app demonstrating how to use the `backend_driven_ui` package.

## WhatsApp Clone Demo

A complete WhatsApp-like UI built entirely from JSON schemas, showcasing:

- Full app structure from backend config (AppBar, Tabs, Content, FAB)
- Server-driven UI rendering with 33+ built-in widgets
- A/B testing support with variant switching
- All UI elements defined in JSON, no hardcoded widgets

## Running the Example

```bash
cd example
flutter pub get
flutter run
```

## Features Demonstrated

- **Backend-Driven UI** - Entire app rendered from JSON config
- **Built-in Widgets** - Text, Column, Row, Container, ListView, Icons, etc.
- **Nested Structures** - Complex layouts with deep widget trees
- **Dynamic Content** - Chats, calls, status updates from JSON
- **Server Control** - Backend controls the entire UI structure
